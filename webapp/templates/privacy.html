Privacy Policy

This Resume Screening Application is built for educational purposes only.

  • No data is stored: All uploaded resumes are processed in memory and not saved permanently.
  • No third-party sharing: Your data is not shared with any external services.
  • Temporary processing: Resume contents are used only during active screening and discarded afterward.
  • Open source nature: This app is intended as a learning tool for machine learning and Flask web development.

If you do not agree with the terms above, please refrain from using this application.
